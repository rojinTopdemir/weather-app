# Full-Stack Hava Durumu Uygulaması

dinamik hava durumu verilerini anlık olarak çeken ve kullanıcı geçmişini kaydeden bir web uygulaması.

##  Özellikler
- Dinamik Arka Plan: Hava durumuna göre (Güneşli, Bulutlu, Yağmurlu) değişen arka plan renkleri ve özel CSS animasyonları.
-Kalıcı Hafıza: Backend tarafında aramaları `.json` dosyasına kaydederek "Son Aramalar" listesi oluşturma.
- Modern UI: Glassmorphism (cam efekti) tasarımı ile şık arayüz.
- REST API: OpenWeatherMap API ile gerçek zamanlı veri entegrasyonu.

## Kullanılan Teknolojiler
- Frontend:HTML5, CSS3 (Advanced Animations), JavaScript (ES6+), Fetch API
- Backend: Node.js, Express.js
- Kütüphaneler:Axios, CORS, File System (FS)
